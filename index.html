Caesar Cipher

The Caesar cipher shifts each letter by a fixed number of positions in the alphabet. For example, with a shift of 3, 'A' becomes 'D'.

How it works: Each letter is shifted by the same amount. Key range: 1-25

Vigenere Cipher

The Vigenere cipher uses a keyword to shift letters at different positions. Each letter of the keyword determines the shift for the corresponding letter in the plaintext.

How it works: Each letter is shifted by an amount determined by the corresponding letter in the key. Key: alphabetic characters only.
Decrypted Output:
-

Hill Cipher

The Hill cipher uses a matrix as the key to encrypt plaintext. It processes multiple letters at once using matrix multiplication modulo 26.

How it works: Uses 2x2 matrix for encryption. Enter 4 numbers (a,b,c,d) for the matrix [[a,b],[c,d]]. Determinant must be coprime with 26.
Decrypted Output:
-

Rail Fence Cipher

The Rail Fence cipher (also called Zigzag cipher) arranges the plaintext in a zigzag pattern across multiple rails, then reads it row by row to create the ciphertext.

How it works: Text zigzags across a specified number of rails. Higher rail count = more security.

-

Columnar Transposition Cipher

The Columnar Transposition cipher rearranges plaintext by writing it in rows under a key, then reading columns in alphabetical order of the key.

How it works: Text is written in rows, columns are rearranged based on alphabetical order of key. Key must be unique letters.
